Main Concepts

The Way the Web Works

The world wide web is essentially a vast collection of html documents dispersed among a myriad of computers and communicated via HTTP Request from the user to a server. The server then retrieves the appropriate html document and returns it to the user's computer, wherein the web browser interprets the document and displays it as a web page on the user's screen. This video makes for a good reference.

HTML

HTML is shorthand for Hypertext Markup Language. HTML documents comprise the majority of the content on the web. HTML documents contain text content, "what you see" and markup, "how it looks". This video makes for a good reference.

Tags and Elements

HTML documents are composed of HTML elements. When writing html, the type of each element is conveyed to the browser by using HTML tags. This video makes for a good reference.

Why Computers are Stupid

This is to say, computers are stupid because they are only able to interpret instructions literally. Therefore, if upon writing a program a small mistake is made, a very big problem may present within the program at runtime.

Inline vs Block Elements

HTML elements are either inline or block. Block elements each occupy a full line and encapsulate their content with an 'invisible box', whereas inline elements only occupy a fraction of a line per element, to the extent that the remainder of space in such line is sufficient.
